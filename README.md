# electricity-bill-system-c
C program to calculate electricity bill and late fee using if-else conditions.


# Electricity Bill System in C

This C program calculates:
- electricity bill
- late fee
- final bill

based on:
- electricity units
- payment status

using if-else conditions.

## Features
- Multiple unit pricing rules
- Late fee system
- Final bill calculation

## Concepts Used
- if-else
- logical conditions
- user input
